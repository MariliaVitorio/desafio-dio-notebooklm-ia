# desafio-dio-notebooklm-ia
Estudo sobre confiabilidade em Inteligência Artificial utilizando NotebookLM como ferramenta de aprendizagem ativa. O projeto investiga limitações, vieses, alucinações e mecanismos de validação para uso ético e responsável da IA, combinando curadoria de fontes, engenharia de prompts e pensamento crítico.
